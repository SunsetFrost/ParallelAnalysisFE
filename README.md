# Parallel Analysis FE

### 功能模块
- Home 主页 
- Cluster Server 集群查看，管理，监控
  - Status 查看集群服务器信息（角色分布， 运行状态）
  - Task（并行计算任务）


    - 查看task信息，运行状态，包含的agent
 
    - 增加，减少 agent（计算节点）
 
    - 查看对应结果数据

- 监控（日志，记录）
  - master
  - agent
  - framework
  - group
- 数据中心
  - 列表
  - 详情+下载
- Jupyter在线计算与报告
  - 数据分析
      - NetCDF数据 可视化
      - MongoDB站点数据可视化
  - 模型并行运算
